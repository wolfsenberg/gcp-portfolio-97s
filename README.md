# GCP Portfolio Capstone Repository

Welcome to the official repository for **Cloud Solutions Department, 2025-2026*. This project serves as a comprehensive capstone repository guide for cadets under the Google Developer Group PUP to learn, deploy, and monitor their personal portfolio using Google Cloud Platform (GCP). Over a 30-day period, cadets will transition from cloud novices to practitioners by building and deploying a basic static portfolio website.

## Usage
You should use this template to start your Capstone project. It contains all of the code that you will need to get started.

Do Not fork this code! It is meant to be used by pressing the Use this Template button in GitHub. This will copy the code to your own repository with no connection back to the original repository like a fork would. This is what you want.

---

## 🗺️ Repository Structure

To prevent information overload, this repository is organized into logical stages:

*   📂 **`01-Setup/`**: Project Initialization, Billing configuration, and IAM roles.
*   📂 **`02-Deployment/`**: Guides for Cloud Run, App Engine, and Static Hosting.
*   📂 **`03-Storage-CDN/`**: Configuring Cloud Storage buckets, Load Balancing, and CDN.
*   📂 **`04-Monitoring/`**: Setting up Cloud Monitoring, Logging, and Uptime Alerts.
*   📂 **`05-Troubleshooting/`**: FAQ and common error resolutions.
*   📂 **`samples/`**: "Plug-and-play" boilerplate code for quick deployment.
